Requirements for this project:
 This project does require the following pip-installable libraries:
     pandas
     matplotlib
     vaderSentiment
     seaborn
     wordcloud

it also uses the following Python Standard Library modules:
datetime
csv
json
os

To install required packages, run:
    pip install pandas matplotlib vaderSentiment seaborn wordcloud

also visit the kagell link to download the data and save it in as IMDB_REVIEW the same codespace as the project
